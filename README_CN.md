# Console Control (控制台控制)

一个用于控制系统控制台窗口的 Blender 插件。

> **仅限 Windows** - 此插件使用 Windows API，仅支持 Windows 系统。

## 功能

- **位置和大小控制** - 自定义控制台窗口的位置和尺寸
- **透明度** - 调整控制台窗口透明度 (10-100%)
- **置顶显示** - 保持控制台窗口在其他窗口之上
- **快捷键切换** - 使用 `Ctrl + `` 快速切换（可在偏好设置中自定义）
- **窗口同步** - 控制台随 Blender 窗口自动最小化/还原
- **启动时自动打开** - 可选在 Blender 启动时自动打开控制台
- **18 种语言** - 支持中文、英文、日文、韩文、法文、德文、西班牙文、葡萄牙文、俄文、意大利文、荷兰文、波兰文、土耳其文、阿拉伯文、泰文、越南文、印尼文、乌克兰文

## 安装

1. 下载或克隆此仓库
2. 将整个文件夹放入 Blender 插件目录：
   - `%APPDATA%\Blender Foundation\Blender\<版本>\scripts\addons\`
   - 或便携版：`<blender>\scripts\addons\`
3. 在 Blender 中启用：编辑 → 偏好设置 → 插件 → 搜索 "Console Control"

## 使用

- **面板**：3D视图 → 侧边栏 (N) → Console 标签
- **快捷键**：`Ctrl + `` 切换控制台显示
- **偏好设置**：编辑 → 偏好设置 → 插件 → Console Control → 自定义快捷键和语言

## 系统要求

- Blender 4.2+
- Windows 10/11
