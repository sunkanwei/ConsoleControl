# Console Control

[中文说明](README_CN.md)

A Blender addon for controlling the system console window.

> **Windows Only** - This addon uses Windows API and only works on Windows.

## Features

- **Position & Size Control** - Set custom position and dimensions for the console window
- **Opacity** - Adjust console window transparency (10-100%)
- **Always On Top** - Keep console window above other windows
- **Hotkey Toggle** - Quick toggle with `Ctrl + `` (customizable in preferences)
- **Window Sync** - Console automatically minimizes/restores with Blender
- **Auto Open** - Optionally open console on Blender startup
- **18 Languages** - English, Chinese, Japanese, Korean, French, German, Spanish, Portuguese, Russian, Italian, Dutch, Polish, Turkish, Arabic, Thai, Vietnamese, Indonesian, Ukrainian

## Installation

1. Download or clone this repository
2. Place the entire folder in your Blender addons directory:
   - `%APPDATA%\Blender Foundation\Blender\<version>\scripts\addons\`
   - Or for portable: `<blender>\scripts\addons\`
3. Enable in Blender: Edit → Preferences → Add-ons → Search "Console Control"

## Usage

- **Panel**: View3D → Sidebar (N) → Console tab
- **Hotkey**: `Ctrl + `` to toggle console visibility
- **Preferences**: Edit → Preferences → Add-ons → Console Control → customize hotkey and language

## Requirements

- Blender 4.2+
- Windows 10/11
